Getting Started

AutoFlask is a powerful Flask application generator that helps you create modern web applications quickly and efficiently.

Installation

pip install autoflask

Configuration

Configure your AutoFlask application by creating a config.py file in your project root. 

SECRET_KEY = 'your-secret-key'
DATABASE_URI = 'sqlite:///app.db'
DEBUG = True

Usage

Create a new Flask application with AutoFlask: 

from autoflask import create_app

app = create_app()
app.run()

How to Use

Create a new Flask project.

positional arguments:
  name                 Name of the project

options:
  -h, --help           show this help message and exit
  --venv-true          Create a virtual environment
  --venv-false         Do not create a virtual environment
  --cat CAT            Category of project (Web Development, Machine Learning, etc.)
  --libs LIBS          Additional libraries to install (comma-separated)
  --install-true       Install selected libraries
  --install-false      Do not install selected libraries
  --location LOCATION  Project location (defaults to current directory)
  --auth               Include Flask Authentication
  --css-tailwind       Use Tailwind CSS
  --css-bootstrap      Use Bootstrap CSS
  --css-none           Do not use any CSS framework
